Express.js используется как основа для API каждого микросервиса.
Sequelize управляет моделями и миграциями для PostgreSQL.
bcrypt применяется в Auth Service для хэширования паролей.
JWT используется для аутентификации (токены в заголовках запросов).
PostgreSQL хранит данные, каждый микросервис имеет свою схему.
Docker контейнеризирует микросервисы для независимого развертывания.
Redis кэширует часто запрашиваемые данные (профили, посты, теги).
Kafka обеспечивает асинхронное взаимодействие (например, отправка уведомлений при лайке).
WebSocket в Messaging Service для real-time чатов.
